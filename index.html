JAGUARETE KAA S.A
(TIENDA E-COMMERCE )

Implementada con:
JDIANGO , HTML Y CSS

Es una pagina web de una tienda de productos. Con un diseño adaptable a mobile, tablet o escritorio.

Hay tres tipos de usarios:

Usuario comun: Agregar productos al carrito.
Usuario moderador: Ingresar productos nuevos.
Administrador: Control de todo.

Para realizar alguna acción en el sitio, debe loguearse primero. Segun que tipo de usuario el sistema identifique, se mostrará las distintas pantallas con sus respectivas acciones.
